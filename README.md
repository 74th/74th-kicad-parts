# 74th の使う KiCad Parts

- symbols: KiCad のシンボル
- footprints: KiCad のフットプリント
- 3d-models: ダウンロードした 3D モデルを置くディレクトリ
- pcb-default: 基板の設定の標準データ。最初にインポートして使う

## 他によく使う KiCad Parts

- Espressif KiCad Library https://github.com/espressif/kicad-libraries
